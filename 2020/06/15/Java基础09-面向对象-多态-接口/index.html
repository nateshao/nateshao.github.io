    09.01_йқўеҗ‘еҜ№иұЎ(еӨҡжҖҒзҡ„жҰӮиҝ°еҸҠе…¶д»Јз ҒдҪ“зҺ°)

    • A:еӨҡжҖҒ(polymorphic)жҰӮиҝ°
      • дәӢзү©еӯҳеңЁзҡ„еӨҡз§ҚеҪўжҖҒ
    • B:еӨҡжҖҒеүҚжҸҗ
      • a:иҰҒжңүз»§жүҝе…ізі»гҖӮ
      • b:иҰҒжңүж–№жі•йҮҚеҶҷгҖӮ
      • c:иҰҒжңүзҲ¶зұ»еј•з”ЁжҢҮеҗ‘еӯҗзұ»еҜ№иұЎгҖӮ
    • C:жЎҲдҫӢжј”зӨә
      • д»Јз ҒдҪ“зҺ°еӨҡжҖҒ

    09.02_йқўеҗ‘еҜ№иұЎ(еӨҡжҖҒдёӯзҡ„жҲҗе‘ҳи®ҝй—®зү№зӮ№д№ӢжҲҗе‘ҳеҸҳйҮҸ)

    • жҲҗе‘ҳеҸҳйҮҸ

    09.03_йқўеҗ‘еҜ№иұЎ(еӨҡжҖҒдёӯзҡ„жҲҗе‘ҳи®ҝй—®зү№зӮ№д№ӢжҲҗе‘ҳж–№жі•)

        09.04_йқўеҗ‘еҜ№иұЎ(еӨҡжҖҒдёӯзҡ„жҲҗе‘ҳи®ҝй—®зү№зӮ№д№ӢйқҷжҖҒжҲҗе‘ҳж–№жі•)

    • йқҷжҖҒж–№жі•
      • зј–иҜ‘зңӢе·Ұиҫ№(зҲ¶зұ»)пјҢиҝҗиЎҢзңӢе·Ұиҫ№(зҲ¶зұ»)гҖӮ
      • (йқҷжҖҒе’Ңзұ»зӣёе…іпјҢз®—дёҚдёҠйҮҚеҶҷпјҢжүҖд»ҘпјҢи®ҝй—®иҝҳжҳҜе·Ұиҫ№зҡ„)
      • еҸӘжңүйқһйқҷжҖҒзҡ„жҲҗе‘ҳж–№жі•,зј–иҜ‘зңӢе·Ұиҫ№,иҝҗиЎҢзңӢеҸіиҫ№

    • A:жЎҲдҫӢеҲҶжһҗ
      • йҖҡиҝҮиҜҘжЎҲдҫӢеё®еҠ©еӯҰз”ҹзҗҶи§ЈеӨҡжҖҒзҡ„зҺ°иұЎ

        09.06_йқўеҗ‘еҜ№иұЎ(еӨҡжҖҒдёӯеҗ‘дёҠиҪ¬еһӢе’Ңеҗ‘дёӢиҪ¬еһӢ)

    • A:жЎҲдҫӢжј”зӨә
      • иҜҰз»Ҷи®Іи§ЈеӨҡжҖҒдёӯеҗ‘дёҠиҪ¬еһӢе’Ңеҗ‘дёӢиҪ¬еһӢ
        Person p = new SuperMan();еҗ‘дёҠиҪ¬еһӢ

        09.07_йқўеҗ‘еҜ№иұЎ(еӨҡжҖҒзҡ„еҘҪеӨ„е’ҢејҠз«Ҝ)

    • A:еӨҡжҖҒзҡ„еҘҪеӨ„
      • a:жҸҗй«ҳдәҶд»Јз Ғзҡ„з»ҙжҠӨжҖ§(з»§жүҝдҝқиҜҒ)
      • b:жҸҗй«ҳдәҶд»Јз Ғзҡ„жү©еұ•жҖ§(з”ұеӨҡжҖҒдҝқиҜҒ)
    • B:жЎҲдҫӢжј”зӨә
      • еӨҡжҖҒзҡ„еҘҪеӨ„
      • еҸҜд»ҘеҪ“дҪңеҪўејҸеҸӮж•°,еҸҜд»ҘжҺҘж”¶д»»ж„Ҹеӯҗзұ»еҜ№иұЎ
    • C:еӨҡжҖҒзҡ„ејҠз«Ҝ
      • дёҚиғҪдҪҝз”Ёеӯҗзұ»зҡ„зү№жңүеұһжҖ§е’ҢиЎҢдёәгҖӮ
    • D:жЎҲдҫӢжј”зӨә
      method(Animal a)
      method(Cat c)

    09.09_йқўеҗ‘еҜ№иұЎ(жҠҪиұЎзұ»зҡ„жҰӮиҝ°еҸҠе…¶зү№зӮ№)

    • A:жҠҪиұЎзұ»жҰӮиҝ°
      • жҠҪиұЎе°ұжҳҜзңӢдёҚжҮӮзҡ„
    • B:жҠҪиұЎзұ»зү№зӮ№
      • a:жҠҪиұЎзұ»е’ҢжҠҪиұЎж–№жі•еҝ…йЎ»з”Ёabstractе…ій”®еӯ—дҝ®йҘ°
        • abstract class зұ»еҗҚ {}
        • public abstract void eat();
      • b:жҠҪиұЎзұ»дёҚдёҖе®ҡжңүжҠҪиұЎж–№жі•пјҢжңүжҠҪиұЎж–№жі•зҡ„зұ»дёҖе®ҡжҳҜжҠҪиұЎзұ»жҲ–иҖ…жҳҜжҺҘеҸЈ
      • c:жҠҪиұЎзұ»дёҚиғҪе®һдҫӢеҢ–йӮЈд№ҲпјҢжҠҪиұЎзұ»еҰӮдҪ•е®һдҫӢеҢ–е‘ў?
        • жҢүз…§еӨҡжҖҒзҡ„ж–№ејҸпјҢз”ұе…·дҪ“зҡ„еӯҗзұ»е®һдҫӢеҢ–гҖӮе…¶е®һиҝҷд№ҹжҳҜеӨҡжҖҒзҡ„дёҖз§ҚпјҢжҠҪиұЎзұ»еӨҡжҖҒгҖӮ
      • d:жҠҪиұЎзұ»зҡ„еӯҗзұ»
        • иҰҒд№ҲжҳҜжҠҪиұЎзұ»
        • иҰҒд№ҲйҮҚеҶҷжҠҪиұЎзұ»дёӯзҡ„жүҖжңүжҠҪиұЎж–№жі•
    • C:жЎҲдҫӢжј”зӨә
      • жҠҪиұЎзұ»зү№зӮ№B:жҠҪиұЎзұ»зү№зӮ№
      • a:жҠҪиұЎзұ»е’ҢжҠҪиұЎж–№жі•еҝ…йЎ»з”Ёabstractе…ій”®еӯ—дҝ®йҘ°
        • abstract class зұ»еҗҚ {}
        • public abstract void eat();
      • b:жҠҪиұЎзұ»дёҚдёҖе®ҡжңүжҠҪиұЎж–№жі•пјҢжңүжҠҪиұЎж–№жі•зҡ„зұ»дёҖе®ҡжҳҜжҠҪиұЎзұ»жҲ–иҖ…жҳҜжҺҘеҸЈ
      • c:жҠҪиұЎзұ»дёҚиғҪе®һдҫӢеҢ–йӮЈд№ҲпјҢжҠҪиұЎзұ»еҰӮдҪ•е®һдҫӢеҢ–е‘ў?
        • жҢүз…§еӨҡжҖҒзҡ„ж–№ејҸпјҢз”ұе…·дҪ“зҡ„еӯҗзұ»е®һдҫӢеҢ–гҖӮе…¶е®һиҝҷд№ҹжҳҜеӨҡжҖҒзҡ„дёҖз§ҚпјҢжҠҪиұЎзұ»еӨҡжҖҒгҖӮ
      • d:жҠҪиұЎзұ»зҡ„еӯҗзұ»
        • иҰҒд№ҲжҳҜжҠҪиұЎзұ»
        • иҰҒд№ҲйҮҚеҶҷжҠҪиұЎзұ»дёӯзҡ„жүҖжңүжҠҪиұЎж–№жі•

    09.10_йқўеҗ‘еҜ№иұЎ(жҠҪиұЎзұ»зҡ„жҲҗе‘ҳзү№зӮ№)

    • A:жҠҪиұЎзұ»зҡ„жҲҗе‘ҳзү№зӮ№
      • a:жҲҗе‘ҳеҸҳйҮҸпјҡж—ўеҸҜд»ҘжҳҜеҸҳйҮҸпјҢд№ҹеҸҜд»ҘжҳҜеёёйҮҸгҖӮabstractжҳҜеҗҰеҸҜд»Ҙдҝ®йҘ°жҲҗе‘ҳеҸҳйҮҸ?дёҚиғҪдҝ®йҘ°жҲҗе‘ҳеҸҳйҮҸ
      • b:жһ„йҖ ж–№жі•пјҡжңүгҖӮ
        • з”ЁдәҺеӯҗзұ»и®ҝй—®зҲ¶зұ»ж•°жҚ®зҡ„еҲқе§ӢеҢ–гҖӮ
      • c:жҲҗе‘ҳж–№жі•пјҡж—ўеҸҜд»ҘжҳҜжҠҪиұЎзҡ„пјҢд№ҹеҸҜд»ҘжҳҜйқһжҠҪиұЎзҡ„гҖӮ
    • B:жЎҲдҫӢжј”зӨә
      • жҠҪиұЎзұ»зҡ„жҲҗе‘ҳзү№зӮ№
    • C:жҠҪиұЎзұ»зҡ„жҲҗе‘ҳж–№жі•зү№жҖ§пјҡ
      • a:жҠҪиұЎж–№жі• ејәеҲ¶иҰҒжұӮеӯҗзұ»еҒҡзҡ„дәӢжғ…гҖӮ
      • b:йқһжҠҪиұЎж–№жі• еӯҗзұ»з»§жүҝзҡ„дәӢжғ…пјҢжҸҗй«ҳд»Јз ҒеӨҚз”ЁжҖ§гҖӮ

    09.11_йқўеҗ‘еҜ№иұЎ(и‘өиҠұе®қе…ё)

    • жЎҲдҫӢжј”зӨә
      • жҠҪиұЎзұ»зҡ„дҪңз”Ё

        09.12_йқўеҗ‘еҜ№иұЎ(жҠҪиұЎзұ»з»ғд№ зҢ«зӢ—жЎҲдҫӢ)

    • A:жЎҲдҫӢжј”зӨә
      • е…·дҪ“дәӢзү©пјҡзҢ«пјҢзӢ—
      • е…ұжҖ§пјҡе§“еҗҚпјҢе№ҙйҫ„пјҢеҗғйҘӯ
      • зҢ«зҡ„зү№жҖ§:жҠ“иҖҒйј 
      • зӢ—зҡ„зү№жҖ§:зңӢе®¶

    09.13_йқўеҗ‘еҜ№иұЎ(жҠҪиұЎзұ»з»ғд№ иҖҒеёҲжЎҲдҫӢ)

    • A:жЎҲдҫӢжј”зӨә
      • е…·дҪ“дәӢзү©пјҡеҹәзЎҖзҸӯиҖҒеёҲпјҢе°ұдёҡзҸӯиҖҒеёҲ
      • е…ұжҖ§пјҡе§“еҗҚпјҢе№ҙйҫ„пјҢи®ІиҜҫгҖӮ
      • е…·дҪ“дәӢзү©:еҹәзЎҖзҸӯеӯҰз”ҹ,е°ұдёҡзҸӯеӯҰз”ҹ
      • е…ұжҖ§:е§“еҗҚ,е№ҙйҫ„,еӯҰд№ 

    09.14_йқўеҗ‘еҜ№иұЎ(жҠҪиұЎзұ»з»ғд№ е‘ҳе·ҘжЎҲдҫӢ)

    • A:жЎҲдҫӢжј”зӨә
      • еҒҮеҰӮжҲ‘д»¬еңЁејҖеҸ‘дёҖдёӘзі»з»ҹж—¶йңҖиҰҒеҜ№зЁӢеәҸе‘ҳзұ»иҝӣиЎҢи®ҫи®ЎпјҢзЁӢеәҸе‘ҳеҢ…еҗ«3дёӘеұһжҖ§пјҡе§“еҗҚгҖҒе·ҘеҸ·д»ҘеҸҠе·Ҙиө„гҖӮ
      • з»ҸзҗҶпјҢйҷӨдәҶеҗ«жңүзЁӢеәҸе‘ҳзҡ„еұһжҖ§еӨ–пјҢеҸҰдёәиҝҳжңүдёҖдёӘеҘ–йҮ‘еұһжҖ§гҖӮ
      • иҜ·дҪҝз”Ёз»§жүҝзҡ„жҖқжғіи®ҫи®ЎеҮәзЁӢеәҸе‘ҳзұ»е’Ңз»ҸзҗҶзұ»гҖӮиҰҒжұӮзұ»дёӯжҸҗдҫӣеҝ…иҰҒзҡ„ж–№жі•иҝӣиЎҢеұһжҖ§и®ҝй—®гҖӮ

    09.15_йқўеҗ‘еҜ№иұЎ(жҠҪиұЎзұ»дёӯзҡ„йқўиҜ•йўҳ)

    • A:йқўиҜ•йўҳ1
      • дёҖдёӘжҠҪиұЎзұ»еҰӮжһңжІЎжңүжҠҪиұЎж–№жі•пјҢеҸҜдёҚеҸҜд»Ҙе®ҡд№үдёәжҠҪиұЎзұ»?еҰӮжһңеҸҜд»ҘпјҢжңүд»Җд№Ҳж„Ҹд№ү?
      • еҸҜд»Ҙ
      • иҝҷд№ҲеҒҡзӣ®зҡ„еҸӘжңүдёҖдёӘ,е°ұжҳҜдёҚи®©е…¶д»–зұ»еҲӣе»әжң¬зұ»еҜ№иұЎ,дәӨз»ҷеӯҗзұ»е®ҢжҲҗ
    • B:йқўиҜ•йўҳ2
      • abstractдёҚиғҪе’Ңе“Әдәӣе…ій”®еӯ—е…ұеӯҳ

    09.16_йқўеҗ‘еҜ№иұЎ(жҺҘеҸЈзҡ„жҰӮиҝ°еҸҠе…¶зү№зӮ№)

    • A:жҺҘеҸЈжҰӮиҝ°

      • д»ҺзӢӯд№үзҡ„и§’еәҰи®Іе°ұжҳҜжҢҮjavaдёӯзҡ„interface
      • д»Һе№ҝд№үзҡ„и§’еәҰи®ІеҜ№еӨ–жҸҗдҫӣи§„еҲҷзҡ„йғҪжҳҜжҺҘеҸЈ

    • B:жҺҘеҸЈзү№зӮ№

      • a:жҺҘеҸЈз”Ёе…ій”®еӯ—interfaceиЎЁзӨә
        • interface жҺҘеҸЈеҗҚ {}
      • b:зұ»е®һзҺ°жҺҘеҸЈз”ЁimplementsиЎЁзӨә
        • class зұ»еҗҚ implements жҺҘеҸЈеҗҚ {}
      • c:жҺҘеҸЈдёҚиғҪе®һдҫӢеҢ–
        • йӮЈд№ҲпјҢжҺҘеҸЈеҰӮдҪ•е®һдҫӢеҢ–е‘ў?
        • жҢүз…§еӨҡжҖҒзҡ„ж–№ејҸжқҘе®һдҫӢеҢ–гҖӮ
      • d:жҺҘеҸЈзҡ„еӯҗзұ»
        • a:еҸҜд»ҘжҳҜжҠҪиұЎзұ»гҖӮдҪҶжҳҜж„Ҹд№үдёҚеӨ§гҖӮ
        • b:еҸҜд»ҘжҳҜе…·дҪ“зұ»гҖӮиҰҒйҮҚеҶҷжҺҘеҸЈдёӯзҡ„жүҖжңүжҠҪиұЎж–№жі•гҖӮ(жҺЁиҚҗж–№жЎҲ)

    • C:жЎҲдҫӢжј”зӨә

      • жҺҘеҸЈзү№зӮ№

        09.17_йқўеҗ‘еҜ№иұЎ(жҺҘеҸЈзҡ„жҲҗе‘ҳзү№зӮ№)

    • A:жҺҘеҸЈжҲҗе‘ҳзү№зӮ№

      • жҲҗе‘ҳеҸҳйҮҸпјӣеҸӘиғҪжҳҜеёёйҮҸпјҢе№¶дё”жҳҜйқҷжҖҒзҡ„е№¶е…¬е…ұзҡ„гҖӮ
          * й»ҳи®Өдҝ®йҘ°з¬Ұпјҡpublic static final
  * е»әи®®пјҡиҮӘе·ұжүӢеҠЁз»ҷеҮәгҖӮ
      • жһ„йҖ ж–№жі•пјҡжҺҘеҸЈжІЎжңүжһ„йҖ ж–№жі•гҖӮ
      • жҲҗе‘ҳж–№жі•пјҡеҸӘиғҪжҳҜжҠҪиұЎж–№жі•гҖӮ
          * й»ҳи®Өдҝ®йҘ°з¬Ұпјҡpublic abstract
  * е»әи®®пјҡиҮӘе·ұжүӢеҠЁз»ҷеҮәгҖӮ

    • B:жЎҲдҫӢжј”зӨә

      • жҺҘеҸЈжҲҗе‘ҳзү№зӮ№

        09.18_йқўеҗ‘еҜ№иұЎ(зұ»дёҺзұ»,зұ»дёҺжҺҘеҸЈ,жҺҘеҸЈдёҺжҺҘеҸЈзҡ„е…ізі»)

    • A:зұ»дёҺзұ»,зұ»дёҺжҺҘеҸЈ,жҺҘеҸЈдёҺжҺҘеҸЈзҡ„е…ізі»

      • a:зұ»дёҺзұ»пјҡ
        • з»§жүҝе…ізі»,еҸӘиғҪеҚ•з»§жүҝ,еҸҜд»ҘеӨҡеұӮз»§жүҝгҖӮ
      • b:зұ»дёҺжҺҘеҸЈпјҡ
        • е®һзҺ°е…ізі»,еҸҜд»ҘеҚ•е®һзҺ°,д№ҹеҸҜд»ҘеӨҡе®һзҺ°гҖӮ
        • е№¶дё”иҝҳеҸҜд»ҘеңЁз»§жүҝдёҖдёӘзұ»зҡ„еҗҢж—¶е®һзҺ°еӨҡдёӘжҺҘеҸЈгҖӮ
      • c:жҺҘеҸЈдёҺжҺҘеҸЈпјҡ
        • з»§жүҝе…ізі»,еҸҜд»ҘеҚ•з»§жүҝ,д№ҹеҸҜд»ҘеӨҡз»§жүҝгҖӮ

    • B:жЎҲдҫӢжј”зӨә

      • зұ»дёҺзұ»,зұ»дёҺжҺҘеҸЈ,жҺҘеҸЈдёҺжҺҘеҸЈзҡ„е…ізі»

        09.19_йқўеҗ‘еҜ№иұЎ(жҠҪиұЎзұ»е’ҢжҺҘеҸЈзҡ„еҢәеҲ«)

    • A:жҲҗе‘ҳеҢәеҲ«

      • жҠҪиұЎзұ»пјҡ
        • жҲҗе‘ҳеҸҳйҮҸпјҡеҸҜд»ҘеҸҳйҮҸпјҢд№ҹеҸҜд»ҘеёёйҮҸ
        • жһ„йҖ ж–№жі•пјҡжңү
        • жҲҗе‘ҳж–№жі•пјҡеҸҜд»ҘжҠҪиұЎпјҢд№ҹеҸҜд»ҘйқһжҠҪиұЎ
      • жҺҘеҸЈпјҡ
        • жҲҗе‘ҳеҸҳйҮҸпјҡеҸӘеҸҜд»ҘеёёйҮҸ
        • жҲҗе‘ҳж–№жі•пјҡеҸӘеҸҜд»ҘжҠҪиұЎ

    • B:е…ізі»еҢәеҲ«

      • зұ»дёҺзұ»
        • з»§жүҝпјҢеҚ•з»§жүҝ
      • зұ»дёҺжҺҘеҸЈ
        • е®һзҺ°пјҢеҚ•е®һзҺ°пјҢеӨҡе®һзҺ°
      • жҺҘеҸЈдёҺжҺҘеҸЈ
        • з»§жүҝпјҢеҚ•з»§жүҝпјҢеӨҡз»§жүҝ

    • C:и®ҫи®ЎзҗҶеҝөеҢәеҲ«

      • жҠҪиұЎзұ» иў«з»§жүҝдҪ“зҺ°зҡ„жҳҜпјҡвҖқis aвҖқзҡ„е…ізі»гҖӮжҠҪиұЎзұ»дёӯе®ҡд№үзҡ„жҳҜиҜҘз»§жүҝдҪ“зі»зҡ„е…ұжҖ§еҠҹиғҪгҖӮ
      • жҺҘеҸЈ иў«е®һзҺ°дҪ“зҺ°зҡ„жҳҜпјҡвҖқlike aвҖқзҡ„е…ізі»гҖӮжҺҘеҸЈдёӯе®ҡд№үзҡ„жҳҜиҜҘз»§жүҝдҪ“зі»зҡ„жү©еұ•еҠҹиғҪгҖӮ

    09.20_йқўеҗ‘еҜ№иұЎ(зҢ«зӢ—жЎҲдҫӢеҠ е…Ҙи·ій«ҳеҠҹиғҪеҲҶжһҗеҸҠе…¶д»Јз Ғе®һзҺ°)

    • A:жЎҲдҫӢжј”зӨә
      • еҠЁзү©зұ»пјҡе§“еҗҚпјҢе№ҙйҫ„пјҢеҗғйҘӯпјҢзқЎи§үгҖӮ
      • зҢ«е’ҢзӢ—
      • еҠЁзү©еҹ№и®ӯжҺҘеҸЈпјҡи·ій«ҳ
    иҪ¬иҪҪиҜ·жіЁжҳҺжқҘжәҗпјҢж¬ўиҝҺеҜ№ж–Үз« дёӯзҡ„еј•з”ЁжқҘжәҗиҝӣиЎҢиҖғиҜҒпјҢж¬ўиҝҺжҢҮеҮәд»»дҪ•жңүй”ҷиҜҜжҲ–дёҚеӨҹжё…жҷ°зҡ„иЎЁиҫҫгҖӮеҸҜд»ҘеңЁдёӢйқўиҜ„и®әеҢәиҜ„и®әпјҢд№ҹеҸҜд»ҘйӮ®д»¶иҮі 1210331079@qq.com
