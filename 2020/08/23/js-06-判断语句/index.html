JavaScript switch 语句

请使用 switch 语句来选择要执行的多个代码块之一。

  • 语法
  • 工作原理：
    1. 首先设置表达式 n（通常是一个变量）。
    2. 随后表达式的值会与结构中的每个 case 的值做比较。
    3. 如果存在匹配，则与该 case 关联的代码块会被执行。
    4. 请使用 break 来阻止代码自动地向下一个 case 运行。
